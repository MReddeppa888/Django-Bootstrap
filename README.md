# 🌍 Tourism Web Application (Django + Bootstrap)

## 📌 Project Overview

This project is a **Tourism Web Application** developed using **Django** and **Bootstrap**.
It showcases multiple cities like **Hyderabad, Warangal, Chittoor, and Kadapa** with information about:

* 🏛️ Famous Places
* 🍛 Local Food
* 🎭 Culture

Each page is designed using **Bootstrap components** and **static images** for a clean and responsive UI.

---

## 🚀 Features

* 📱 Fully responsive design using Bootstrap
* 🖼️ Static images for fast loading and offline usage
* 🌆 Multiple city pages:

  * Hyderabad
  * Warangal
  * Chittoor
  * Kadapa
* 🧭 Simple navigation between sections (Places, Food, Culture)
* ⚡ Clean UI with cards, navbar, and hero sections

---

## 🛠️ Technologies Used

* **Frontend:** HTML5, Bootstrap 5
* **Backend:** Django (Python)
* **Static Files:** Images stored locally

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/your-username/your-repo-name.git
```

2. Navigate to the project folder:

```
cd your-repo-name
```

3. Run the Django server:

```
python manage.py runserver
```

4. Open browser and enter:

```
http://127.0.0.1:8000/
```

---

## 🌐 Accessing Pages

You can access different pages by entering paths in the browser:

```
/hyderabad
/warangal
/chittoor
/kadapa
```

👉 When you enter a URL path, Django routes it to the corresponding HTML page and displays the content.

---

## 💡 How It Works

* Django handles **URL routing**
* Each URL maps to a **view**
* Views render **HTML templates**
* Templates use **Bootstrap + static images**
